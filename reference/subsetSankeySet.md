# Apply subset index on a sankey set

Apply subset index on a sankey set

## Usage

``` r
subsetSankeySet(x, set_id, index = list())
```

## Arguments

- x:

  giottoSankeyPlan object

- set_id:

  set index or node label

- index:

  new index subset, provided in the same order as the set_id

## Value

a `giottoSankeyPlan`
